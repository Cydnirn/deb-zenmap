# deb-zenmap
Script I've made for fun to install zenmap on debian based distro

## Prequisites
Execute using root or user with sudo access

## How to
1. Add execute mode to script
```bash
sudo chmod +x install_zenmap.sh
```
2. Execute the script
```bash
./install_zenmap.sh
```
